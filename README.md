# AviasalesProject
Задачи:\
●	Прогнозирование цен на авиабилеты. Задача включается в себя также определение факторов, от которых зависит цена билета.\
●	Определение самых популярных маршрутов.\
●	Определение средней цены по маршрутам.\
●	Выявление наличия корреляции между расстоянием перелета и ценой билета.\●	Выявление самых коротких и самых продолжительных перелетов.\
