# AviasalesProject
Задачи:\
●	Прогнозирование цен на авиабилеты. Задача включается в себя также определение факторов, от которых зависит цена билета.\
●	Определение самых популярных маршрутов.\
●	Определение средней цены по маршрутам.\
●	Выявление наличия корреляции между расстоянием перелета и ценой билета.\
●	Выявление самых коротких и самых продолжительных перелетов.

Модули:
1.	Парсер данных на языке программирования Python.
2.	База данных. Представляет собой общий и вспомогательные csv-файлы, который используется при анализе данных и построении визуализаций.
3.	Модуль визуализации.
